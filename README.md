# C. Edit Shop — FakeStore E-Commerce App

## Overview
C. Edit Shop is a fully responsive e-commerce frontend built with React, 
React Router, Axios, and React Bootstrap. It connects to FakeStoreAPI 
to display, add, edit, and delete products dynamically.

## Features
- 🛍️ Browse all products with category filtering
- 📄 View detailed product information
- ➕ Add new products via form
- ✏️ Edit existing products
- 🗑️ Delete products with confirmation modal
- 🛒 Add to cart interaction
- 📱 Fully responsive on all screen sizes

## Tech Stack
- React
- React Router
- Axios
- React Bootstrap
- Vite

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open `localhost:5173` in your browser

## API
This app uses [FakeStoreAPI](https://fakestoreapi.com) — a free mock 
e-commerce API for testing and prototyping.
